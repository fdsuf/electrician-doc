<summary>2025/11/06：故障修完走半道又被叫回来，就是长期莫名跳闸，最后查出的原因出乎意料！</summary>
  
## 故障现象
- 设备（数数机）漏电保护器频繁跳闸
- 跳闸时间不固定（半小时至1小时）
- 即使设备不启动也会跳闸
- 某个接线端子有明显发热

## 故障原因
**主线鼻虚接导致接触不良**
- 接线端子压接不实，接触电阻过大发热
- 发热氧化加剧虚接，绝缘下降产生漏电（漏电流821mA）
- 初期误判为设备漏电，实际是连接点问题

## 解决方法
1. **分段检测**：用摇表逐路测量绝缘电阻
2. **定位发热点**：发现接线端子温度异常
3. **重新压接**：拆下端子清理后重新紧固
4. **结果**：温度恢复正常，漏电电流降至80mA，故障排除

## 经验教训
- 频繁跳闸需检查线路连接点而不仅是绝缘
- 虚接发热是隐蔽故障，肉眼不易察觉
- 检修要结合温度检测和物理状态检查

## 故障现象
- 电路中部分区域（"拉拢那边"）没电
- 火线带电，零线也带电（危险现象）
- 局部电路完全断电

## 故障原因
- **主要原因**：零线接线端子螺丝未拧紧，导致虚接
- **直接原因**：零线断路，使断点后侧的零线通过电器与火线等电位
- **根本原因**：前期施工不规范，接线工作不细致

## 解决方法
1. **安全断电**：先切断电源，用电笔验证无电
2. **系统排查**：打开配电箱，逐一检查所有接线端子
3. **紧固螺丝**：找到松动的零线接线点，用螺丝刀彻底拧紧
4. **功能测试**：恢复供电后验证零线不再带电，故障区域恢复供电

## 经验教训
- **施工规范**：电工接线必须每个螺丝都紧固到位，不能有任何疏忽
- **安全意识**：零线带电是重大安全隐患，必须立即停电排查
- **排查方法**：从配电箱开始系统性检查，往往问题出在简单的连接点上
- **质量责任**：施工质量直接影响用电安全，必须认真负责
